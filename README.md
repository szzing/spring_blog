# 🌸 spring_blog

## 1. 프로젝트 개요
회원 관리, 게시판 기능을 구현하는 방법을 연습하기 위해 **스프링 프레임 워크를 이용하여 MVC(Model2 Type) 패턴을 적용한 블로그**를 제작하였다.
